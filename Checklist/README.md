O que é necessário para aprender REACT?
R: HTML, CSS e JavaScript(ES6+) -> promises, arrow functions, destructuring

-O que é REACT?
R: Uma biblioteca utilizada para criar interfaces para o usuário. 
É possivel fazer para web, mobile entre outros.

-React Components:
As aplicações são feitas por COMPONENTES, como se fossem peças de lego. As peças criadas no projeto podem ser utilizadas várias vezes durante o projeto.
O componente nada mais é que uma função JavaScript(JSX)

-Comando para instalar a biblioteca react(O projeto deve estar sem nenhum arquivo):
R: npx create-react-app .
